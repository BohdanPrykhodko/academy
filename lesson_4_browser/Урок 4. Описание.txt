Настройка sublime text 3 для молниеносной верстки
- Плагин emmet
- Установка node.js
- Установка browser sync

Домашнее задание:
1. Создать папку с сайтом со следующией структурой внутри.
— Папка css в которой находится файл style.css
— Папка fonts - в ней будут храниться шрифты для нашего сайта
— Папка js
— Папка img
Файл index.html - это главный файл нашего сайта, проследите чтобы он не виглядел как index.html.txt

2. Разметить документ index.html как это сделано в уроке.
Используйте теги header, nav, section, article, footer.
Прочитайте на webref.ru/html про теги head, body, meta-теги, header, section, nav, article, footer.

3. Установить node.js и browser-sync для дальнейшей работы.
Сайты:
nodejs.org
browsersync.io

Для тех, у кого Windows:
1) Переместите файл bs.bat в папку с сайтом;
2) Откройте bs.bat;
3) Browser-Sync запустится
